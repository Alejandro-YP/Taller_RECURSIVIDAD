Taller de Recursividad en Java

Este repositorio contiene la solución al taller de recursividad.

El proyecto está dividido en 14 programas:

1. Factorial de un número
2. Invertir un número
3. Sumatoria: 1 + 1/2 + 1/3 + ... + 1/n
4. Suma de los dígitos de un número
5. Sumatoria hasta n
6. Potencia (base^exponente)
7. Máximo Común Divisor (MCD) - Algoritmo de Euclides
8. Copia de una cadena
9. División entera por restas sucesivas
10. Multiplicación por sumas sucesivas
11. Suma de elementos de un arreglo
12. Suma de una matriz
13. Serie de Fibonacci
14. Función de Ackermann
